# Description
This is a node todo tool that I developed myself, serving a similar purpose to a memo or a reminder.

# Install

```
yarn global add node-todo-zlh

```


# Usage

```
td 
td add <message>
td read
td remove
td -h
```


# The usage of the script file is as follows:
```
node cli
node cli add
node cli read
node cli remove
```
